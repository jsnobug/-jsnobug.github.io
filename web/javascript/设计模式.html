# 设计模式

学习用高内聚，低耦合的封装代码。

new作用：

  1. 创建一个新对象；
  2. 将构造函数的作用域赋给新对象(this指向了这个新对象)
  3. 执行构造函数中的代码(为这个新对象添加属性);
  4. 返回新对象
  • 能用健壮的代码去解决具体的问题；
  • 能用抽象的思维去应对复杂的系统；
  • 能用工程化的思想去规划更大规模的业务。
